Instruction for testing the example code for the iss node_module:

1. Make sure you have node.js installed. Please find the directions for installing node.js on your operating system here:
https://nodejs.org/en/

2. unpack this directory into a location of your choosing

3. to test an example program open your command line prompt and navigate to the javaScript example directory of your choice.

4. type 'npm install'. npm will install the required node modules as specified by the package.json file.

5. once modules are installed type 'node [exampleProgram].js

6. The program should run and report back the parsed data in your command line with a description header.
